完整功能清单

  • 诊断
    慢接口 Top N 排名

    按耗时倒序，每条标注 5 类原因标签（后端慢 / 网络拥塞 / 首次连接 / 浏览器排队 / 响应过大）

  • 诊断
    前后端责任判定

    利用 Resource Timing + 服务端 extend.runtime 交叉验证，客观区分谁的问题

  • 检测
    业务层失败抓取

    HTTP 200 但 JSON 业务状态码异常，自动识别并展示响应体详情

  • 检测
    HTTP 状态异常

    4xx / 5xx / 超时等非 200 接口单独列出，含响应体片段

  • 评估
    网络环境 0-100 评分

    综合 navigator.connection + 域名实测 RTT，得出客观网络质量分数

  • 评估
    Web Vitals 体验指标

    LCP / FCP / CLS / INP / TTFB / Long Tasks，用好/中/差三档直观展示

  • 追踪
    用户操作轨迹

    最近 30 次点击/输入/路由跳转/页面切换，带时间戳和元素定位。不记录密码。

  • 追踪
    JS 异常 + 资源失败

    window.onerror、unhandledrejection、link/script/img 加载失败，完整错误栈

  • 架构
    iframe 多框架支持

    后台管理系统常用 iframe 嵌套，Judge 自动采集所有子框架数据并合并分析

  • 导出
    PDF / 摘要 / 完整报告

    一键生成带 IP 水印的 PDF；复制文字摘要粘到工单；打开网页版完整交互报告

  • 告警
    被动 Badge 提醒

    页面出现接口异常或 JS 报错时，扩展图标自动显示红/琥珀色角标计数

  • 检测
    域名 Ping 实测

    对页面涉及的每个 API 域名发送探测请求，实测 RTT 和连通性
