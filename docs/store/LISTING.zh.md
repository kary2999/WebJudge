## 详细说明(支持基本格式,16384 字符以内)

```
Judge 是一款页面性能诊断工具,装在 Chrome 上,在你觉得网站"慢"的时候点一下,
就能告诉你到底是哪里慢、谁的责任 —— 是后端处理、网络拥塞、前端排队,还是
你自己的网络环境差。它帮非技术同事提交"可被立即定位"的工单,而不是一句
"系统好慢啊"。

═══════ 它能帮你 ═══════

✓ 慢接口 Top 排名,按耗时倒序
✓ 区分前后端责任 — 5 类标签:后端慢 / 网络拥塞 / 首次连接慢 / 浏览器排队 / 响应过大
✓ 网络环境 0-100 评分 — 客观告诉你是不是"你网慢"
✓ 抓 HTTP 异常 + 业务层失败(HTTP 200 但响应 status ≠ 0)
✓ 监听 JS 运行时异常 — 解释"页面白了"的根因
✓ 记录最近 30 个用户操作 — 同事忘了刚才点了什么时,这里能找到
✓ 支持 iframe 后台架构 — 多 frame 数据自动合并
✓ 一键导出带 IP 水印的完整 PDF 报告

═══════ 适合谁用 ═══════

• 公司后台系统的非技术使用者(运营 / 产品 / 客服)
• 经常被人吐槽"系统好慢但说不清"的开发同学
• 需要给开发提交"可定位"工单的任何角色

═══════ 隐私承诺 ═══════

• 所有分析在你浏览器本地完成
• 不向任何外部服务器发送你浏览的数据
• 唯一外部请求:可选的公网 IP 查询(用于 PDF 水印,可关闭)
• 不读 cookie / form 输入 / 密码框
• 关闭标签页数据立即释放
• 代码完全开源可审

═══════ 怎么用 ═══════

1. 装上扩展
2. 打开"慢"的网页,按 F5 刷新一次
3. 复现卡顿操作
4. 点工具栏放大镜图标
5. 看弹窗诊断,或下载完整 PDF 报告

═══════ 开发者备注 ═══════

Judge 不会自动上传任何信息。如发现工具栏图标上出现红色/琥珀色数字角标,
那是被动监听到了页面异常,提示你"刚刚有 N 次接口/JS 错误"。打开弹窗即清零。

完整文档与源码请参考开发者主页 / 开源仓库。
```
